# Audio Classification: Predicting Song Genres
Audio data is quickly becoming an important part of machine learning. From using audio to recommend songs for radio channels, all the way to interacting with virtual assistants like Alexa or Siri and helping self-driving cars hear their surroundings rather than just seeing them.
